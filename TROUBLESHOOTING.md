# 🔧 حل المشاكل الشائعة

## ❌ المشكلة: Uncaught SyntaxError أو ReferenceError

### السبب:
هذه الأخطاء تحدث عندما يحاول النظام استخدام مكتبة Supabase قبل أن تكون جاهزة.

### ✅ الحل (تم تطبيقه):

لقد تم تحديث الملفات لحل هذه المشكلة:

#### 1. تم تعديل `config.js`:
```javascript
// بدلاً من:
const supabase = window.supabase?.createClient(...)

// أصبح:
let supabase = null; // سيتم تهيئته لاحقاً
```

#### 2. تم إضافة دالة `initSupabase()` في `app.js`:
- تتحقق من تحميل المكتبة أولاً
- تتحقق من وجود المفاتيح
- تنشئ الاتصال في الوقت المناسب

## 🐛 الأخطاء الشائعة وحلولها

### 1. ❌ "Supabase is not defined"

**السبب**: المكتبة لم تُحمّل

**الحل**:
```html
<!-- تأكد من وجود هذا السطر في <head> في index.html -->
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
```

### 2. ❌ "Invalid API key"

**السبب**: المفتاح خطأ أو قديم

**الحل**:
1. اذهب لـ Supabase Dashboard
2. Settings → API
3. انسخ المفتاح مرة أخرى
4. الصقه في `config.js`

### 3. ❌ "Table 'companies' does not exist"

**السبب**: لم تنفذ SQL script

**الحل**:
1. افتح Supabase Dashboard
2. SQL Editor
3. انسخ محتوى `database-setup.sql`
4. الصقه واضغط Run

### 4. ❌ "Failed to fetch"

**السبب**: مشكلة في الاتصال بالإنترنت أو CORS

**الحل**:
- تحقق من اتصالك بالإنترنت
- تأكد من صحة Project URL
- حاول تحديث الصفحة

### 5. ❌ "Cannot upload file"

**السبب**: Storage bucket غير موجود

**الحل**:
1. Supabase Dashboard → Storage
2. Create bucket: `receipts`
3. اجعله Public ✓
