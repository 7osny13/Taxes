## 🌐 خطوات الرفع (الطريقة السهلة)

- [ ] دخلت على [GitHub.com](https://github.com)
- [ ] ضغطت على زر "+" في الأعلى
- [ ] اخترت "New repository"
- [ ] كتبت اسم: `tax-invoice-tracker`
- [ ] اخترت Public
- [ ] **ما اخترتش** "Add README"
- [ ] ضغطت "Create repository"

### 2️⃣ رفع الملفات
- [ ] في الصفحة الجديدة، ضغطت "Upload files"
- [ ] رفعت الملفات التالية:

**الملفات الأساسية (4):**

- [ ] style.css
- [ ] app.js
- [ ] config.js

**ملفات الاختبار (2):**
- [ ] test.html
- [ ] simple-test.html

**ملفات التوثيق (6):**

**ملفات أخرى:**
- [ ] database-setup.sql
- [ ] GITHUB-UPLOAD-GUIDE.md

- [ ] كتبت Commit message: "Initial commit"
- [ ] ضغطت "Commit changes"

### 3️⃣ تفعيل GitHub Pages
- [ ] فتحت Settings في الـ Repository
- [ ] ضغطت على "Pages" من القائمة الجانبية
- [ ] اخترت Branch: `main`
- [ ] اخترت Folder: `/ (root)`
- [ ] ضغطت "Save"
- [ ] انتظرت دقيقة وظهر الرابط الأخضر
- [ ] نسخت الرابط: `https://username.github.io/tax-invoice-tracker/`
