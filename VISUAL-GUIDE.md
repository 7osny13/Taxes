## 🌟 الخطوة 1: إنشاء Repository

### 1.1 افتح GitHub.com

```
┌─────────────────────────────────────────┐
│  🌐 https://github.com                  │
│                                         │
│  ┌────────────────┐  ┌──────────────┐  │
│  │   Sign in      │  │   Sign up    │  │
│  └────────────────┘  └──────────────┘  │
└─────────────────────────────────────────┘
```

**افتح المتصفح واذهب إلى GitHub**

---

### 1.2 اضغط على زر "+"

```
┌─────────────────────────────────────────┐
│  GitHub     🔍 Search        [+] ▼  😊  │
│                               ↑          │
│                          اضغط هنا       │
│  ┌─────────────────────────────────┐    │
│  │  New repository                 │    │
│  │  Import repository              │    │
│  │  New gist                       │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

**اختر "New repository"**

---

### 1.3 املأ بيانات الـ Repository

```
┌─────────────────────────────────────────────┐
│  Create a new repository                    │

│                                             │
│  ◉ Public    ○ Private                     │
│                                             │

│                                             │
│  ┌──────────────────────┐                  │
│  │ Create repository    │                  │
│  └──────────────────────┘                  │
└─────────────────────────────────────────────┘
```

**املأ كما في الصورة واضغط "Create repository"**
